### Цель

Создать GitHub Actions, который будет собирать приложение, запускать тесты и публиковать docker образ.

---

### Описание

Ссылка на [GitHub](https://github.com/OTUS-NET/ASP.NET/blob/main/Homeworks/06%20Настройка%20CI/Task.md)

Перед выполнением нужно ознакомиться с [Правилами отправки домашнего задания на проверку](https://github.com/OTUS-NET/ASP.NET/blob/main/Homeworks/docs/homework-rules.md)

1. Запуск тестов в Dockerfile
   - Добавить в Dockerfile стейдж test, который будет запускать тесты PromoCodeFactory.UnitTests.csproj
2. Cоздать Github Actions
   - Создать файл ci.yml по пути .github/workflows (в корне репозитория)
   - Настроить сборку и публикацию docker образа
3. Настройка триггера
   - воркфлоу должен запускаться, если произошёл коммит в ветку main и изменения были папке 'Homeworks/06 Настройка CI/**'
   - Также можно добавить возможность ручного запуска воркфлоу
4. Проверка работы опубликованного образа
   - Добавить docker-compose.yaml, который использует опубликованный в ghcr образ
